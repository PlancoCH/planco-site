## Purpose

This repository contains the portfolio Site for Planco.

## Description

This project is the main marketing and product website for Planco, built with React. It serves as the primary public-facing presence of the company, showcasing Planco's features, values, and offerings. The site is designed to be fast, responsive, and easily maintainable, with a component-based architecture that allows for straightforward iteration as the product evolves.

## Prerequisites

Before you begin, ensure you have the following installed:

* Node.js (LTS)
* npm or yarn

## Installation & Setup

Step-by-step instructions to get the development environment running.

```bash
# Clone the repository
git clone git@github.com:PlancoCH/planco-site.git

# Install dependencies
npm install

# Set up environment variables
cp .env.example .env
```

## Usage

Examples of how to run the project locally or build for production.

```bash
# Start development server
npm run dev

# Build for production
npm run build

# Preview production build locally
npm run preview
```
